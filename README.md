## Intro

**Pinnect AI**는 영상을 내용으로 검색하고, 관련된 영상을 제공합니다. 영상을 통해 정보를 얻고 싶은 사용자들은 **쉽고∙빠르게∙신뢰성 높은 자료를 제공**받을 수 있습니다. 지금 [웹사이트](https://www.pinnect-ai.com/)와 [크롬 웹 스토어](https://chrome.google.com/webstore/detail/pinnectai-easy-fast-and-a/dgehmmodngcdaiaoomenancidjnnfpac)에서 지금 바로 시작하세요.

## Installation
### Download
```bash
git clone https://github.com/brain-pathfinder/frodo.git
cd frodo
```
### Environment Variables
* .env을 편집합니다.
  ```bash
  vi .env
  ```
* [.env] 값을 그대로 복사하고 저장합니다.

### Run Docker
* 도커이미지를 실행시킵니다.
  ```bash
  docker compose up --build -d
  ```

### Reference
* 더 자세한 서버설정은 [프로도 노션페이지]를 참고하세요.
